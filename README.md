# plm-numlock

---

Plasma Login Manager自动启用小键盘

本文档同时提供[*Typst*](./README.typ)版本


[**English**](./README_en.md) | [**中文简体**](./README.md)

# 用途
使用 Plasma Login Manager 作为系统的显示管理器（登录管理器）时，自动在登陆界面启用数字小键盘

# 使用方法
```bash
git clone https://github.com/bailuocf/plm-numlock.git
cd ~/plm-numlock
./plm-numlock.sh
```
